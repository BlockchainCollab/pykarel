pyKarel
=======

A framework for teaching students new to programming about various programming
concepts, using the python language.

Coding:
-------

pyKarel is designed to make it as easy as possible for the student using the
library. All initialization is done with "import pykarel". This has some
technical shortcomings but has the advantage of making labwork extremely
simple.
